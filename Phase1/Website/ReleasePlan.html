ACME Release Plan

Exploration Phase

On Saturday (20th September), we met as a group to discuss and plan our project. We started by going over both librarian and researcher user stories. Together we came up with 25 user stories combined. We then took time to group user stories into features that could be developed over the same time period. We used HIGH/MEDIUM/LOW method to prioritize features described in the 25 user stories. In the Commitment Phase we have come up with our first date, in which we plan to develop initial stages of our application based on various user stories. Every Phase we will reiterate by improving our application and adding new features using the agile method.



Commitment Phase
